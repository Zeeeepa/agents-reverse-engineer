|----------|------------------|-----------------|-----------|
| JavaScript/TypeScript | `package.json` | npm, yarn, pnpm | ✓ |
| Python | `requirements.txt`, `pyproject.toml`, `setup.py`, `Pipfile` | pip, poetry, pipenv | ✓ |
| Ruby | `Gemfile` | bundler | |
| Go | `go.mod` | go modules | ✓ |
| Rust | `Cargo.toml` | cargo | ✓ |
| Java | `pom.xml` (Maven), `build.gradle` (Gradle) | Maven, Gradle | |
| Kotlin | `build.gradle.kts`, `build.gradle` | Gradle | |
| C#/.NET | `*.csproj`, `packages.config`, `*.fsproj` | NuGet | |
| PHP | `composer.json` | Composer | |
| Swift | `Package.swift` | Swift PM | |
| Elixir | `mix.exs` | Mix | |
| Erlang | `rebar.config` | rebar3 | |
| Scala | `build.sbt` | sbt | |
| Clojure | `deps.edn`, `project.clj` | clj, Leiningen | |
| Haskell | `package.yaml`, `*.cabal`, `stack.yaml` | cabal, stack | |
| Dart/Flutter | `pubspec.yaml` | pub | |
| Lua | `*.rockspec` | LuaRocks | |
| R | `DESCRIPTION` | CRAN | |
| Julia | `Project.toml` | Pkg | |
